Projects

Clustering Analysis of Indonesian Provinces Based on Rice Farming Characteristics in 2024: Harvested Area, Productivity, and Production

This project involves a clustering analysis of Indonesian provinces based on three key rice farming characteristics in 2024: harvested area, productivity, and production volume. We applied the K-Means clustering algorithm to group provinces with similar agricultural profiles. The goal is to identify patterns and regional similarities in rice cultivation, which can support data-driven decision-making in agricultural planning and policy development.

Application of Naive Bayes Method in Classifying Countries Based on Financial and Economic Indicators

This project applies the Naive Bayes classification method to group countries based on selected financial and economic indicators. By using supervised learning, the model aims to classify countries into categories that reflect their economic conditions. The project demonstrates how probabilistic models like Naive Bayes can be effectively used for macroeconomic analysis and global financial profiling.

Second-Order Dynamic System Analysis on Tractor Suspension Using a Mass-Spring Model to Support Plowing Efficiency in the Food Security Program

This project analyzes a second-order dynamic system on tractor suspension using a mass-spring model, aiming to improve the efficiency of plowing in rice fields. By modeling the mechanical behavior of the suspension system, we evaluate how vibration and stability affect tractor performance. The findings contribute to optimizing agricultural machinery as part of efforts to enhance productivity and support national food security programs.

Forecasting Global Oil Prices in 2025 Using the ARIMA Model

This project utilizes the ARIMA (AutoRegressive Integrated Moving Average) model to forecast global oil prices for the year 2025. By analyzing historical time series data, the model captures trends and patterns to generate accurate price predictions. The results aim to support economic planning and decision-making in sectors affected by oil market fluctuations.

Hypothesis Testing and Parameter Estimation of Bitcoin Prices

This project focuses on conducting statistical hypothesis tests and parameter estimation on Bitcoin price data. By applying inferential statistical methods, we aim to determine key characteristics such as mean price behavior, volatility patterns, and confidence intervals—providing a deeper understanding of Bitcoin's market dynamics and supporting data-driven investment insights.

Development of SIGINTA (Transaction Intelligence System) Application for Detecting Suspicious Transactions in the Digital Financial Sector Using K-Means Clustering

This project involves the development of SIGINTA, a transaction intelligence application designed to detect suspicious activities in the digital financial sector. By implementing the K-Means clustering algorithm, the system groups transaction data to identify unusual patterns that may indicate fraudulent or high-risk behavior. SIGINTA aims to enhance early detection and support financial institutions in improving risk management and regulatory compliance.
